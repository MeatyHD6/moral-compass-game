## How It Works

1. **Scenario Engine** - JavaScript array stores all ethical dilemmas
2. **Choice Tracking** - Records which framework player chooses each time
3. **Results Calculator** - Counts choices and generates personalized summary
4. **Visual Feedback** - CSS animations provide immediate response to choices
